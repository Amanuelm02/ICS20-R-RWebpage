Client Server vs Peer to Peer

What are they?

Both Client-Server and Peer-to-Peer networks connect computers so that resources like files and applications can be shared.
Client-server networks have one or more central computers, or servers, that hold the data and manage resources.Peer-to-peer networks connect computers so that each computer shares all or part of its resources.

Client Sever

  • Has a specific server and specific clients connected to the server
  • The client request for service and server respond with the service
  • Shares the Information
  • The client request for service and server respond with the service
  • The data is stored in a centralized server
  • The client-server are expensive to implement
  • Client-Server is more stable and scalable
  • When several clients request for the services simultaneously, a server can get bottlenecked

    • Peer-to-Peer

  • Clients and server are not distinguished; each node act as client and server
